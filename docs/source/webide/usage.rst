WebIDE 使用
==================

WebIDE 是 XiaoHeiHook 的电脑端脚本开发界面。手机端启动本地 HTTP 服务后，电脑浏览器可以访问并编辑脚本。

启动 WebIDE
-----------------

1. 打开 XiaoHeiHook 设置页。
2. 如需后台稳定运行，先点击“忽略电池优化”。
3. 开启 WebIDE。
4. 推荐绑定地址使用 ``127.0.0.1``，端口默认 ``8787``。
5. 电脑执行：

.. code-block:: bash

   adb forward tcp:8787 tcp:8787

6. 浏览器访问：

.. code-block:: text

   http://127.0.0.1:8787/

.. warning::
   WebIDE 可以编辑和同步 Hook 脚本。绑定到局域网地址或 ``0.0.0.0`` 可能让同一网络中的其他设备访问脚本编辑入口，请谨慎开启。

界面区域
-----------------

WebIDE 使用朴素的传统 IDE 布局：

.. code-block:: text

   顶部工具栏：保存 / 同步 / 重启 / 同步并重启
   左侧面板：软件列表 / 全部脚本
   中间区域：Monaco 多标签编辑器
   右侧面板：Hook 设置 / 脚本设置
   底部终端：实时日志与同步结果

快捷键
-----------------

.. list-table::
   :header-rows: 1
   :widths: 30 70

   * - 快捷键
     - 说明
   * - ``Ctrl + S`` / ``Cmd + S``
     - 保存当前脚本。
   * - ``F5``
     - 同步并重启当前应用。
   * - ``Ctrl + Shift + S``
     - 同步当前应用。
   * - ``Ctrl + R``
     - 重启当前应用。

全部脚本页面
-----------------

“全部脚本”页面用于统一管理 ``Documents/XiaoHeiHook`` 下的脚本。鼠标悬停脚本项时会显示手势光标，点击脚本即可在编辑器中打开。

多标签编辑
-----------------

编辑器支持同时打开多个脚本。未保存的标签会显示未保存标记，关闭未保存标签时会弹出确认。

终端单开页面
-----------------

WebIDE 终端支持单独打开页面查看，适合把日志放到另一块屏幕或另一个浏览器窗口中。
